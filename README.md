# ReviewAI Platform

Enterprise AI Code Review Platform for developers and engineering teams.

## Overview

ReviewAI is an AI-powered code review platform consisting of:

- VS Code Extension
- Node.js Backend API
- PostgreSQL
- Redis
- BullMQ
- Kafka
- Docker
- AI Integration

The platform helps engineering teams enforce coding standards, perform AI-powered code reviews, generate fixes, and analyze code quality.

## Features

- AI Code Review
- Company Coding Rules
- Pull Request Review
- AI Auto Fix
- Review History
- Team Analytics
- GitHub Integration
- VS Code Extension
